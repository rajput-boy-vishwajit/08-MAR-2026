<h1>Git & GitHub in One Shot 🚀</h1>

<h3>What is Version Control System?</h3>

<p>
A <b>Version Control System (VCS)</b> is a tool that helps developers track changes in code
and manage different versions of a project.
</p>

<h3>What is Git?</h3>

<p>
<b>Git</b> is a Distributed Version Control System.
</p>

<h3>Features of Git</h3>

<ul>
<li>Popular</li>
<li>Free & Open Source</li>
<li>Fast and scalable</li>
<li>Tracks code changes efficiently</li>
</ul>

<h3>Example Git Command</h3>

<pre>
git status
git add .
git commit -m "message"
</pre>
